How To Run:
1. Extract the gzip file. Look for quant.jar executable.
2. Command to run app
`java -cp quant.jar com.ashish.Main -f /pathToCookieFile/cookie.csv -d 2018-12-09`
3. Code can be browsed at https://github.com/chauha/quantscope
4. Tests can be run as `mvn clean test` after importing the maven project.
